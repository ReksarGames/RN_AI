## 🧠 Обзор

**RN_AI** — это высокопроизводительный AI помощник на основе YOLO с интуитивным интерфейсом **DearPyGui**. 
Построен с использованием **фильтра Калмана**, **алгоритмов SunOner** и оптимизирован для конкурентного гейминга.

![RN_AI Demo](docs/demo.gif)

- **Точка входа:** `main.py`
- **Основная логика:** `src/` и `core/`
- **Вдохновлено:** [SunOner's sunone_aimbot_cpp](https://github.com/SunOner/sunone_aimbot_cpp)

> 💡 *Проект SunOner солидный и хорошо спроектирован, но этот Python pipeline обеспечивает еще более высокий FPS, чем C++.*

## ⚡ Быстрый Старт

### Установка и Настройка

<details open>
<summary><b>Шаг 1: Установка Зависимостей</b></summary>

```batch
install.bat
```
</details>

<details>
<summary><b>Опционально: мастер установки GPU/CPU (installer.py)</b></summary>

```batch
python src/installer.py
```

Что делает:
- **GPU режим**: скачивает установщик CUDA 12.8 в папку репозитория, затем ставит PyTorch CUDA (cu128), TensorRT и onnxruntime-gpu.
- **CPU режим**: ставит onnxruntime-directml.

Примечания:
- Устанавливает Python-пакеты в **то окружение, в котором вы запускаете** (системный Python или venv).
- CUDA ставится **системно** через официальный установщик NVIDIA.
</details>

<details open>
<summary><b>Шаг 2: Запуск Приложения</b></summary>

```batch
run.bat
```
</details>

## 🤖 Поддержка Моделей

RN_AI поддерживает множество форматов и версий моделей YOLO:

| Формат | Поддержка | Предпочтение | Примечания |
|--------|----------|-------------|-----------|
| **ONNX** | ✅ Полная | Стандартный | Кроссплатформная совместимость |
| **TensorRT** (.engine) | ✅ Полная | **Предпочтительно** ⭐ | Лучшая производительность на NVIDIA GPU |

### Поддерживаемые Версии YOLO
- ✅ YOLO v8 по v12
- 📁 Примеры моделей доступны в `.weight/`
- 🎛️ Полностью настраивается из GUI
